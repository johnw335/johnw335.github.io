What is group_by?

I think the best way to describe this method is to look at an example and work through it piece by piece. I found this example on markusjais.com.

cats = ["Tiger", "Lion", "Puma", "Leopard", "Jaguar","Cheetah", "Bobcat"] grouped_by_length = cats.group_by { |cat| cat.length } #{5=>["Tiger"], 4=>["Lion", "Puma"], 7=>["Leopard", "Cheetah"], 6=>["Jaguar", "Bobcat"]}

or

birds = ["Golden Eagle", "Gyrfalcon", "American Robin","Mountain BlueBird", "Mountain-Hawk Eagle"] grouped_by_first_letter = birds.group_by { |s| s[0] } # {"G"=>["Golden Eagle", "Gyrfalcon"], "A"=>["American Robin"], # "M"=>["Mountain BlueBird", "Mountain-Hawk Eagle"]}

Let's focus on that 2nd example and see how group_by works. We start with an array called birds and we decide that we want to group the elements in the array by their first letter. We create a variable called grouped_by_first_letter and sey it equal to birds grouped by { |s| s[0] }. This means it is grouped by the 0 index for each word, or the first letter.

I can see this method being very useful in storing data. Looking back, I can think of pleanty of times I would have already used it during DBC. I'm glad that I have a new method under my belt andf I can't wait to explore even more.
